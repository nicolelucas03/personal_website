Spot That Tune

A music trivia game built in Swift using Spotify's API and SQLite

Developed in Xcode, this iOS app lets users authenticate with Spotify, choose a genre and game mode, and guess songs based on short audio previews. Player scores are stored locally using SQLite and displayed on a dynamic leaderboard, promoting friendly competition. The app delivers an engaging, music-driven experience by combining real-time interaction with seamless Spotify integration.

Exploring Mouse-Based User Identification

A privacy-conscious research project exploring the use of mouse behavior for biometric user identification

This project investigated whether users can be uniquely identified through their mouse movement patterns. We collected raw mouse data from the browser, processed it to reduce noise, and engineered features such as cursor speed, hesitation time, and pathing behavior. By applying clustering and classification models, we assessed the viability of mouse-based tracking as a privacy-preserving alternative to traditional methods like cookies or IP addresses.
